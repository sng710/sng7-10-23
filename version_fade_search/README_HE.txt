גרסת faded leaf מקורית + טעינה מ-assets/people-original

מה יש בגרסה הזו:
- index.html מבוסס על הקובץ המקורי שצורף, עם אותו רקע עלים, מבנה, טיפוגרפיה ותפריט.
- אין צורך ב-data.js.
- מוצגים 6 אנשים בכל סבב.
- הפופאפ לוקח טקסט מ-profile_text.txt ותמונות מתוך photos.
- נתמך נתיב assets/people-original וגם assets/people.

העלאה ל-GitHub:
1. החליפי את index.html בגרסה הזו.
2. השאירי את assets/people-original כמו שהוא.
3. מומלץ להריץ במחשב run_build_manifest_windows.bat ואז להעלות את people_assets_manifest.js שנוצר.
4. אם ה-manifest ריק, הדף ינסה לסרוק את GitHub Pages אוטומטית דרך GitHub API.

מבנה תיקייה נתמך:
